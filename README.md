# Word Puzzle Game (Java)

This is a console-based word puzzle game built using backtracking.

## Features
- 2D board word search
- User input gameplay
- Tracks progress of found words

## Concepts Used
- DFS (Depth First Search)
- Backtracking
- HashSet

## How to Run
- Compile and run the Java file
- Enter words to guess from the board
